# Crisp Agriculture multi-page website

This is a Next.js website prepared for Vercel deployment.

## Pages
- Home
- Services
- Contact

## Local run
```bash
npm install
npm run dev
```

## Deploy with GitHub + Vercel
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In Vercel, choose **Add New Project**.
4. Import the GitHub repository.
5. Click **Deploy**.

## Connect the GoDaddy domain
After deployment in Vercel:
1. Open the project in Vercel.
2. Go to **Settings > Domains**.
3. Add `crispag.org` and `www.crispag.org`.
4. Vercel will show the exact DNS records to place in GoDaddy.
