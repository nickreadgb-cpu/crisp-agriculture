# Crisp Agriculture Premium Site

This is a Vercel-ready Next.js site for Crisp Agriculture.

## Local development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Create a GitHub repository.
2. Upload all files from this folder.
3. In Vercel, import the GitHub repository.
4. Deploy.
5. In Vercel, add your custom domain under **Settings > Domains**.
6. Update your GoDaddy DNS records with the values Vercel provides.
